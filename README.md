# Paradise Nursery E-Commerce Application

An interactive e-commerce shopping cart web application designed for an online plant nursery using React and Redux Toolkit.

## Key Features Implemented
- **Landing Page:** Features a full-screen immersive background with a "Get Started" call-to-action button.
- **Product Grid:** Displays 3 distinct plant collections (Air Purifying, Fragrant, and Succulents) with 6 items per category.
- **State Management:** Integrated with Redux Toolkit for unified action dispatching (`addItem`, `removeItem`, `updateQuantity`).
- **Dynamic Cart Interface:** Disables product buttons instantly upon selection and updates navbar indicators in real time.
